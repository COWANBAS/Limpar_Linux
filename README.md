# O que o script faz

O script tem como intuito fazer uma limpeza completa no sistema removendo pacotes antigos logs caches e o dns da sua distro, assim deixando ela mais leve para o dia a dia.

Cuidado ao utilizar versões iniciais dos scritp pois ele pode remover configurações previas de algums aplicativos como algums pacotes do flatpaks que não são utilizados a um tempo.
